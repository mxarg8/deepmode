**Video Model** — Creates short animated clips from the same image set. It's not full animation — think short expressive clips rather than cinematic sequences. Still in active development, but functional enough to generate reel-style content.

**All-in-One Model** — Combines both capabilities in a single trained model. More expensive upfront in credits, but if you're doing mixed content (stills + motion), you only train once.

For first-timers: start with the Image Model. It's the most proven path to results, and you can always train a separate video model later once you understand how the system works.
